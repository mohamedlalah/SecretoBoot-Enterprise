# Installation Guide

## 1. Extract the ZIP

Extract the release package anywhere on your Windows system.

## 2. Run Installer

Open the `Scripts` folder and run:

```text
Install_SecretoBoot_Enterprise_v7_1.cmd
```

as Administrator.

## 3. Reboot

After installation is complete, reboot your computer.

## 4. Troubleshooting

Run:

```text
Scripts\Check_Active_Config.cmd
```

to verify the active rEFInd configuration.
