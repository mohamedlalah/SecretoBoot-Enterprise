# Code of Conduct

## Our Pledge

We want SecretoBoot Enterprise to be a respectful and helpful project for users, testers, and contributors.

## Expected Behavior

- Be respectful.
- Help others when possible.
- Report issues clearly.
- Stay focused on legal, safe, and educational use.
- Respect third-party software licenses.

## Unacceptable Behavior

- Harassment or abusive language
- Illegal use or promotion
- Sharing malicious code
- Claiming ownership of third-party projects
- Removing credits from SecretoBoot or rEFInd

## Reporting

If you see inappropriate behavior or a project issue, please open a GitHub Issue or contact the project maintainer.
